# PongC
Pong in C, built to learn C with Raylib.

## Glitches and Warning
Bit glitchy although I just made this to learn C in a fun way.
I learnt about structs, vars, strings, functions, libraries, and C operations!
## How to Build
Use the following command to compile (Windows):

```bash
gcc src/main.c -o main -I./include -L./lib -lraylib -lgdi32 -lwinmm
